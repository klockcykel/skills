# Interactive CV
An interactive CV visualized with D3.

[Demo](https://klockcykel.se/skills/)

## Screenshot
![](screenshot.png?raw=true)

## Customize
Add skills to _skills.txt_. Name of the skill, optionally followed by the level of expertise.

## Build
```
npm install
npm run build
```

Deploy the dist folder on any static web server or load locally.
